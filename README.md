# Browser-Extension-Security-Audit-Task7
## Objective
To identify potentially harmful browser extensions, review their permissions, and remove suspicious or unused extensions.

## Tools Used
- Google Chrome Browser

## Steps Performed

1. Opened Chrome extension manager (chrome://extensions/)
2. Reviewed all installed extensions.
3. Checked permissions and site access for each extension.
4. Identified unused or unnecessary extensions.
5. Removed suspicious/unnecessary extension.
6. Restarted browser to observe performance changes.

## Security Learning

Browser extensions can pose security risks if they:
- Access all website data
- Read browsing history
- Modify webpage content
- Collect sensitive information

Always install extensions from trusted developers and review permissions carefully.

## Outcome
Improved awareness of browser security risks and safe extension management.
